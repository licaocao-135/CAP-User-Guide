执行接口测试用例

接口测试用例和脚本编写完成后，您可以执行一次用例，验证用例是否可以正常执行，并调试用例。您也可以在需要的时候，随时执行用例。

操作步骤

  1. 在用例库界面中，单击用例名称，进入用例详情界面。
  2. 单击“测试脚本”页签。
  3. 单击右上方的“运行”。
