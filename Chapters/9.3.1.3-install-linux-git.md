# 安装和配置Linux Git客户端

操作系统为Linux时，根据其发行版本安装Git客户端：
* Debian/Ubuntu系统
    
    在终端中输入以下命令行安装：
```
    $ apt-get install git
```

* Fedora/Centos/Redhat系统
    
    在终端中输入以下命令行安装：
```
    $ yum install git
```

*   更多发行版本的安装指导请参见[Git Linux官网链接](https://git-scm.com/download/linux)。
